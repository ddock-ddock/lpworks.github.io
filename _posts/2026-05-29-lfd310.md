투자 의무 달성 현황 화면은 조합별 투자 의무와 현재 달성 현황을 한 화면에서 조회하는 화면입니다.

조합명, 투자 의무 유형, 의무명, 달성률, 의무/한도 금액, 달성 금액, 잔여 금액 등을 확인할 수 있으며, 조합 또는 투자 의무 상세 화면으로 이동할 수 있습니다.

## 설명

### 화면 경로

`LPworks` → `조합 관리` → `투자 의무 달성 현황`

### 화면 구성
![](/assets/img/lfd310img.png)

### 투자 의무 달성 현황 조회

목록에서는 조합별 투자 의무와 달성 현황을 확인할 수 있습니다.

| 컬럼 | 설명 |
| --- | --- |
| 조합명 | 투자 의무가 설정된 조합명입니다. 조합명을 클릭하면 조합 정보 화면으로 이동합니다. |
| 운용사 | 해당 조합을 운용하는 운용사입니다. |
| 투자의무유형 | 투자 의무의 유형입니다. |
| 의무명 | 투자 의무의 이름입니다. |
| 조건 | 하나의 의무에 여러 조건이 있는 경우 조건 정보가 표시됩니다. |
| 달성률 | `달성 금액 ÷ 의무/한도 금액 × 100`으로 계산된 달성 비율입니다. |
| 의무/한도 금액 | 투자 의무에 설정된 기준 금액입니다. |
| 달성 금액 | 해당 투자 의무 기준으로 실제 투자 집행된 금액의 합계입니다. |
| 잔여 금액 | `의무/한도 금액 - 달성 금액`으로 계산된 잔여 금액입니다. |
| 시작일 | 투자 의무 적용 시작일입니다. |
| 종료일 | 투자 의무 적용 종료일입니다. |

### 상세 화면 이동

투자 의무 달성 현황 목록에서 다음 화면으로 이동할 수 있습니다.

| 동작 | 이동 화면 |
| --- | --- |
| 조합명 클릭 | 해당 조합의 조합 정보 화면으로 이동합니다. |
| 행 클릭 | 해당 투자 의무의 상세 화면으로 이동합니다. |

### 검색 및 필터

각 컬럼 상단의 입력창 또는 필터 아이콘을 사용하여 목록을 검색하거나 조건별로 필터링할 수 있습니다.

| 필터 유형 | 적용 컬럼 |
| --- | --- |
| 텍스트 검색 | 조합명, 운용사, 의무명, 달성률 |
| 선택 필터 | 조합명, 운용사, 투자의무유형, 조건 |
| 숫자 검색 | 의무/한도 금액, 달성 금액, 잔여 금액 |
| 날짜 검색 | 시작일, 종료일 |

### 정렬 기준

목록은 기본적으로 아래 기준에 따라 정렬됩니다.

1. 조합명 내림차순
2. 같은 조합명인 경우 투자 의무 유형 순서 오름차순
3. 같은 투자 의무 유형인 경우 조건 순서 오름차순

## 자주 묻는 질문

> 달성률은 어떻게 계산되나요?
{: .prompt-tip }

- 달성률은 `달성 금액 ÷ 의무/한도 금액 × 100`으로 계산됩니다.

> 잔여 금액은 무엇을 의미하나요?
{: .prompt-tip }

- 잔여 금액은 투자 의무 또는 한도 금액에서 현재까지 달성한 금액을 차감한 금액입니다.

> 조합명을 클릭하면 어디로 이동하나요?
{: .prompt-tip }

- 선택한 조합의 조합 정보 화면으로 이동합니다.

> 목록의 행을 클릭하면 어디로 이동하나요?
{: .prompt-tip }

- 선택한 투자 의무의 상세 화면으로 이동합니다.

> 조건 컬럼은 언제 표시되나요?
{: .prompt-tip }

- 하나의 투자 의무에 여러 조건이 설정된 경우, 각 조건을 구분하기 위해 표시됩니다.
