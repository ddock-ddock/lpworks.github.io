업무집행조합원이 요청한 출자금 납입 내역을 조회하고 납입여부를 관리할 수 있는 출자금 납입 요청 기능에 대한 설명입니다.

![이미지](/assets/img/444.png)
## 설명
- 출자금 납입요청에서는 조합원이 GP로부터 본인에게 전달된 **출자금 납입요청 내역을 조회**하고, **납입 금액·계좌 정보**를 확인할 수 있습니다.

#### 출자금 납입요청 내역 조회

조합원에게 전송된 출자금 납입요청 내역을 조회하는 방법은 다음과 같습니다.

1. LPworks에 로그인해 주세요.
2. `조합-출자/배분-출자금 납입요청` 메뉴로 이동해 주세요.
3. 출자금 납입요청 목록에서 본인에게 전달된 납입요청 내역을 확인할 수 있습니다.
  
목록에서는 다음 정보를 확인할 수 있습니다.

| 항목명      | 설명                                 |
| -------- | ---------------------------------- |
| 운용사      | 출자금 납입요청을 발송한 운용사 정보입니다.           |
| 조합       | 출자 대상 조합명을 확인할 수 있습니다.             |
| 납입 요청일   | 운용사가 출자금 납입요청을 등록한 일자입니다.          |
| 통화       | 출자금 납입 통화(KRW, USD 등)를 확인할 수 있습니다. |
| 금융 기관    | 출자금 납입을 위한 금융기관 정보입니다.             |
| 계좌번호     | 출자금 납입 계좌번호를 확인할 수 있습니다.           |
| 납입 요청 금액 | 해당 회차에 요청된 출자금 납입 금액입니다.           |
| 납입 상태    | 출자금 납입 진행 상태를 확인할 수 있습니다.          |
1. 조합 > 연말정산 소득공제 > 연도별 투자 확인서 메뉴를 클릭해 주세요.
2. 조회하고자 하는 소득공제 내역의 귀속 연도를 선택해 주세요.
3. [조회] 버튼을 클릭해 주세요.
4. 해당 연도의 조합별 소득공제 내역이 출력됩니다.
	- [tip] 운용사에서 해당 연도의 소득공제 내역을 전송한 적이 없는 경우, 리스트에서 해당 연도를 찾을 수 없습니다.

#### 출자금 납입요청 상세 조회

출자금 납입요청 목록에서 특정 행을 클릭하면, 해당 납입요청의 상세 정보를 확인할 수 있습니다.

###### 기본정보 확인 
상세 화면 상단의 **기본 정보 영역**에서는 다음 항목을 확인할 수 있습니다.

| 항목명   | 설명                                  |
| ----- | ----------------------------------- |
| 운용사   | 출자금 납입요청을 발송한 운용사 정보입니다.            |
| 조합    | 출자 대상 조합명을 확인할 수 있습니다.              |
| 통화    | 출자금 납입 통화 정보를 확인할 수 있습니다.           |
| 약정 금액 | 해당 조합에 대한 총 출자 약정 금액입니다.            |
| 납입액   | 현재까지 납입 완료된 금액입니다.                  |
| 약정 잔액 | 약정 금액에서 누적 납입액을 차감한 잔여 출자 가능 금액입니다. |

###### 납입 요청 정보 확인
기본 정보 하단의 **납입 요청 정보 영역**에서는 이번 회차의 출자금 납입요청 상세를 확인할 수 있습니다.

| 항목명      | 설명                                |
| -------- | --------------------------------- |
| 납입 요청 금액 | 이번 회차에 요청된 출자금 납입 금액입니다.          |
| 납입 후 잔액  | 해당 납입요청 금액을 반영한 이후의 약정 잔액입니다.     |
| 금융기관     | 출자금 납입을 위한 금융기관 정보입니다.            |
| 계좌번호     | 출자금 납입 계좌번호입니다.                   |
| 납입 요청일   | 운용사가 납입요청을 등록한 일자입니다.             |
| 납입 상태    | 출자금 납입 진행 상태를 나타냅니다.              |
| 요청 사유    | 운용사가 입력한 출자금 납입요청 사유를 확인할 수 있습니다. |
###### 첨부파일
- 운용사가 출자금 납입요청과 함께 등록한 첨부파일이 있는 경우, 상세 화면 하단에서 확인할 수 있습니다.
- 첨부파일은 다운로드하여 열람할 수 있습니다.
- 첨부파일이 없는 경우, **“첨부파일이 없습니다.”**로 표시됩니다.


#### 납입 상태 안내

- 상세 화면 우측 상단에서 현재 출자금 `납입 상태`를 변경할 수 있습니다. 
- 납입 상태는 조합원 자신의 관리용이며, 실제 납입 여부를 반영하지 않습니다.

| 납입 상태 | 설명                        |
| ----- | ------------------------- |
| 미납    | 출자금 납입이 아직 완료되지 않은 상태입니다. |
| 납입 완료 | 출자금 납입이 완료된 상태입니다.        |

## 자주 묻는 질문

> 출자금 납입은 어디에서 하나요?  
> {: .prompt-tip }

- 출자금 납입요청 화면에서는 납입 계좌 정보만 제공됩니다.
- 실제 출자금 이체는 조합원이 직접 금융기관을 통해 진행해 주세요.
    

> 납입을 완료했는데 상태가 변경되지 않았어요.  
> {: .prompt-tip }

- 납입 상태는 상세 화면에서 [납입 확인]버튼을 클릭하여 직접 변경합니다.
- 실제 납입내역과 연동되지 않습니다.
- 운용사와 무관한 조합원 관리용입니다.
    

> 첨부 파일이 보이지 않아요.  
> {: .prompt-tip }

- 운용사가 첨부파일을 등록하지 않은 경우, 파일이 표시되지 않을 수 있습니다.
- 필요한 자료가 없는 경우 운용사에 문의해 주세요.
    

> 출자금 납입요청 메일을 받지 못했어요.  
> {: .prompt-tip }

- LPworks에 등록된 이메일 주소가 정확한지 확인해 주세요.
- 이메일 정보는 조합원 정보 화면에서 수정할 수 있습니다.
