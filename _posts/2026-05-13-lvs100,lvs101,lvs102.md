업무집행조합원이 전송한 조합의 **투심위 보고** 사항을 조회하고, 해당 보고의 상세 정보를 조회할 수 있는 기능에 대한 설명입니다.

## 설명
- 투심위 보고에서는 조합원이 GP로부터 본인에게 전달된 투자심의위원회(이하 투심위) 건을 조회하고, 해당 보고의 기본 정보 및 투심 상세 정보를 확인할 수 있습니다.

#### 투심위 보고 목록 조회
- 조합원에게 전송된 투심위 보고 목록을 조회하는 방법은 다음과 같습니다.
![이미지](/assets/img/lvs100.jpg)

1. LPworks에 로그인해 주세요.
2. `포트폴리오 > 투심위 보고` 메뉴로 이동해 주세요.
3. 목록에서 본인에게 전달된 투심위 보고 내역을 조합별로 확인할 수 있습니다.
	- 동일 투자기업에 대해 복수의 조합에서 투심위가 진행되더라도, 투심위 건은 조합 단위로 구분됩니다.
  
- 목록에서는 다음 정보를 확인할 수 있습니다.

| 항목명    | 설명                                                                                                                                                                 |
| ------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 투심일자   | 투자심의위원회(이하 투심위)의 개최일자를 확인할 수 있습니다.                                                                                                                                 |
| 투심시간   | 투심위 개최 시간을 확인할 수 있습니다.                                                                                                                                             |
| 조합명    | 투심위의 재원이 되는 조합을 확인할 수 있습니다.                                                                                                                                        |
| 운용사명   | 해당 조합의 운용사명을 확인할 수 있습니다.                                                                                                                                           |
| 투자기업명  | 투자 대상 기업명을 확인할 수 있습니다.                                                                                                                                             |
| 주요제품   | 투자 대상 기업에서 취급하는 주요 제품을 확인할 수 있습니다.                                                                                                                                 |
| 총 투자금액 | 해당 투심위의 총 투자 금액을 확인할 수 있습니다.<br>이는 조합 통화로 환산된 금액입니다.                                                                                                               |
| 투심상태   | 투심위의 상태를 확인할 수 있습니다.<br>`계획` : 해당 투심위가 아직 개최되기 전임을 의미합니다.<br>`가결` : 해당 투심위 결과가 가결되었음을 의미합니다.<br>`부결`: 해당 투심위 결과가 부결되었음을 의미합니다.<br>`검토중단`: 해당 투심위 검토가 중단되었음을 의미합니다. |


#### 투심위 보고 상세 조회
- 투심위 보고 목록에서 특정 행을 클릭하면, 해당 투심위 보고의 상세 정보를 확인할 수 있습니다.

##### 기본정보 탭 
- 상세 화면의 기본 정보 탭에서는 다음 항목을 확인할 수 있습니다.
![이미지](/assets/img/lvs101.jpg)

###### 투심 개요
- 투자심의위원회에 대한 개요 정보를 확인할 수 있습니다.

###### 담당 심사역
- 해당 투심위의 담당 심사역 정보를 확인할 수 있습니다.
	- `발굴`, `심사`, `사후관리`별로 각 담당 심사역(정,부)을 확인할 수 있습니다.

###### 회사 정보
- 투자기업에 관한 정보를 확인할 수 있습니다.

###### 발행 주식 개요
- 투자기업에 대한 투자 전/후의 총 발행 주식수를 확인할 수 있습니다.
	- 각 총 발행 주식수는 보통주와 우선주를 구분해서 확인할 수 있습니다.

##### 투심 정보 탭
- 상세 화면의 투심 정보 탭에서는 다음 항목을 확인할 수 있습니다.
![이미지](/assets/img/lvs102.jpg)

###### 투자 조건
- 해당 투심위에 대한 투자 조건을 확인할 수 있습니다.

| 항목명                  | 설명                                                                                                           |
| -------------------- | ------------------------------------------------------------------------------------------------------------ |
| 조합명                  | 투심위의 재원이 되는 조합입니다.                                                                                           |
| 통화                   | 아래 모든 항목에 적용되는 통화입니다.<br>이는 조합 통화와 동일합니다.                                                                    |
| 총 투자금액               | 해당 투심위의 총 투자 금액입니다.<br>각 투자 유형별 투자금액의 총합을 의미합니다.                                                             |
| 최초 자금 지출일            | 최초 투자금 납입이 이뤄지는 일자입니다.                                                                                       |
| 투자 라운드               | 투자가 진행되는 투자 라운드 정보입니다.                                                                                       |
| 동반 투자 내역             | 본 투자에 동반 투자가 진행되는지 여부입니다.                                                                                    |
| 기업가치(Pre/Post)       | 투자기업에 대한 투자 전/후의 기업가치입니다.                                                                                    |
| 투자 유형                | 투심위에서 심사하는 모든 세부 투자 유형입니다.<br>각 투자 유형별로 셀이 나눠집니다.<br>투자 유형이 주식인 경우, 신주/구주 구분도 같이 표기됩니다.                      |
| 투자 금액                | 해당 투자 유형에 대한 투자금액입니다.                                                                                        |
| 투자 단가                | 해당 투자 유형이 주식일 경우 존재합니다.<br>해당 투자 주식의 주당 투자 금액입니다.                                                            |
| 인수 주식 수              | 해당 투자 유형이 주식일 경우 존재합니다.<br>해당 투자 주식 수입니다.                                                                    |
| 투자 후 지분율             | 해당 투자 유형이 주식일 경우 존재합니다.<br>해당 투자 후 투자 대상 기업에 대한 지분율입니다.                                                      |
| 주목적N 투자 여부           | 해당 투자 유형이 주목적N 투자를 만족하는지 여부입니다.<br>본 화면에서 주목적 투자 여부는 최대 3개까지 표기됩니다.<br>주목적N 자체가 존재하지 않을 경우, '해당 없음'으로 표기됩니다. |
| 특수관계인 거래 여부          | 해당 투자가 특수관계인 거래를 만족하는지 여부입니다.                                                                                |
| ESG 평가 여부            | 해당 투자가 ESG 평가를 만족하는지 여부입니다.                                                                                  |
| EXIT 계획              | 해당 투자에 대한 회수 계획입니다.                                                                                          |
| 투자금 사용 용도            | 투자 대상 기업의 투자금 사용 용도입니다.                                                                                      |
| 의무불이행에 따른 제재 관련 이자율  | 투자 의무를 불이행 했을 경우 발생하는 제재와 관련된 이자율입니다.                                                                        |
| 투자금 사용 용도 위반 시 제재 방안 | 투자 대상 기업이 투자금 사용 용도를 위반했을 경우에 대비한 제재방안입니다.                                                                   |


###### 투자 유형별 투자 조건 상세
- 투자 조건에 존재하는 각 투자 유형 셀을 클릭하면 '투자 조건 상세' 드로어가 등장합니다.
- 드로어에서 투자 유형별로 상이한 상세 투자 조건을 확인할 수 있습니다.
	- 투자 유형에 따라, 드로어 내 섹션 구성이 상이합니다.

| 섹션명          | 항목명          | 설명                                                                                       |
| ------------ | ------------ | ---------------------------------------------------------------------------------------- |
| 투자 유형        | 조합명          | 해당 투자의 재원이 되는 조합명입니다.                                                                    |
|              | 투자 유형        | 해당 투자의 투자 유형입니다.<br>(주식, 채권, 기타)                                                         |
|              | 세부 투자 유형     | 해당 투자의 세부 투자 유형입니다.<br>(보통주, RCPS, BW, CB, SAFE 등)                                       |
|              | 투자 통화        | 해당 투자의 통화입니다.                                                                            |
|              | 투자 금액(투자 통화) | 투자 통화로 표기한 해당 투자 금액입니다.                                                                  |
|              | 적용 환율(1투자통화) | 1단위의 투자통화에 대한 조합통화 환율입니다.                                                                |
|              | 투자 금액(조합 통화) | 조합 통화로 환산한 해당 투자 금액입니다.                                                                  |
| 투자 의무        | 목적           | 조합 규약에 존재하는 투자 의무의 목적입니다.<br>(의무투자, 주목적투자, 특수목적투자 등)                                     |
|              | 의무명          | 해당 투자 의무의 의무명입니다.                                                                        |
|              | 해당 여부        | 해당 투자가 각 투자 의무에 해당하는지 여부입니다.                                                             |
| 기타 투자 조건     | 조건N          | 해당 투자에 속한 기타 투자 조건입니다.                                                                   |
|              | 조건명          | 해당 기타 투자 조건의 유형명입니다.<br>(공동매도참여권, 동반매도청구권 등)                                             |
|              | 조건 설명        | 해당 기타 투자 조건에 대한 상세 설명입니다.                                                                |
| 상세 유형별 투자 조건 |              | 투자 유형에 따라, 필요한 투자 조건 내용입니다.<br>주식, 채권, 전환권, 상환권 등 각 투자 유형에 따라, 상이한 섹션 항목에 대한 조건 및 설명입니다. |

###### 동반 투자 조건
- 해당 투심위 대상 기업에 대해 동반 투자를 진행하는 본 조합 외의 투자에 대한 설명입니다.
	- 각 행을 클릭하여 등장하는 '동반 투자 조건 상세' 드로어를 통해, 해당 동반 투자의 상세 조건도 확인할 수 있습니다(세부 투자 유형, 적용 환율 등)

| 항목명      | 설명                       | 비고                          |
| -------- | ------------------------ | --------------------------- |
| 동반 투자자 명 | 동반 투자자 명입니다.             |                             |
| 투자 유형    | 해당 동반 투자의 투자 유형입니다.      |                             |
| 신주/구주    | 해당 동반 투자 주식의 유형입니다.      | 해당 동반 투자 유형이 주식일 경우만 존재합니다. |
| 통화       | 해당 동반 투자의 투자 통화입니다.      |                             |
| 주당단가     | 해당 동반 투자 주식의 주당 투자금액입니다. | 해당 동반 투자 유형이 주식일 경우만 존재합니다. |
| 주식수      | 해당 동반 투자 주식 수입니다.        | 해당 동반 투자 유형이 주식일 경우만 존재합니다. |
| 투자금액     | 해당 동반 투자의 투자 금액입니다.      | 해당 투자 통화로 표기됩니다.            |
| 지분율      | 해당 동반 투자의 투자 후 지분율입니다.   | 해당 동반 투자 유형이 주식일 경우만 존재합니다. |

###### 첨부파일
- 해당 투심위 보고 전송 시에, GP가 첨부한 관련 자료입니다.
- 각 파일을 클릭하여 내려 받을 수 있습니다.

| 항목명       | 설명                                                                |
| --------- | ----------------------------------------------------------------- |
| 투자심사보고서   | 해당 투심위에 대한 투자심사보고서 파일입니다.                                         |
| 준법사항체크리스트 | 해당 투심위에 대한 준법사항체크리스트 파일입니다.                                       |
| 투심위의사록    | 해당 투심위 결과를 기록한 투심위의사록 파일입니다.<br>투심위 상태가 `가결` 혹은 `부결`인 경우에만 첨부됩니다. |


## 자주 묻는 질문

> 투자 통화와 조합 통화가 다른 경우, 적용 환율은 어느 시점 기준인가요?
> {: .prompt-tip }

- 적용 환율은 GP가 **VCworks의 해당 투심위의 투자 정보를 등록 시 입력**한 값입니다.
    

> 주목적 투자 여부 항목이 '투자 조건 상세' 드로어에서 더 많이 등장해요.
> {: .prompt-tip }

- 본 화면의 투자 조건 섹션에서는 조회의 편의를 위하여 조합에 등록된 **주목적 투자 여부만 총 3개까지** 보여주고 있습니다.
- 실제 해당 투자의 모든 투자 의무 해당 여부 조회를 희망하실 경우, 각 투자 유형을 클릭하여 등장하는 '**투자 조건 상세**' **드로어**의 **투자 의무** 섹션을 통해 확인하실 수 있습니다.
    

> `투자 단가`, `투자 후 지분율` 등 비어있는 셀이 있어요.
> {: .prompt-tip }

- 투자 조건 및 동반 투자 조건 섹션에서는 투자 유형이 '**주식**'에 포함되는 경우에만, `투자 단가(주당단가)`, `인수 주식 수(주식수)`, `투자 후 지분율(지분율)`의 값이 표기됩니다.
