업무집행조합원이 전송한 조합의 기준일별 **재무제표**를 조회할 수 있는 기능에 대한 설명입니다.

## 설명
- 재무제표에서는 조합원이 GP로부터 전달받은 조합의 기준일별 재무상태표와 손익계산서 주요 계정 값을 조회하고, 각 보고서 파일을 내려받을 수 있습니다.

#### 재무제표 목록 조회

- 조합원에게 전송된 조합 재무제표 목록을 조회하는 방법은 다음과 같습니다.
![이미지](/assets/img/lfd600.jpg)

1. LPworks에 로그인해 주세요.
2. `조합관리 > 재무제표` 메뉴로 이동해 주세요.
3. 목록에서 본인에게 전달된 재무제표의 기본 정보 및 주요 계정 값을 조합-기준일 별로 확인할 수 있습니다.
  
- 목록에서는 다음 정보를 확인할 수 있습니다.

| 항목명   | 설명                                  |
| ----- | ----------------------------------- |
| 조합 상태 | 해당 조합의 운용, 청산 등의 상태를 확인할 수 있습니다.    |
| 조합명   | 해당 조합명을 확인할 수 있습니다.                 |
| 운용사명  | 해당 조합의 운용사명을 확인할 수 있습니다.            |
| 기준일   | 해당 조합 재무제표의 기준일 및 기준기간을 확인할 수 있습니다. |
| 결산월   | 해당 조합 회계연도의 결산월을 확인할 수 있습니다.        |
| 회계기준  | 해당 조합 회계원장의 회계기준을 확인할 수 있습니다.       |
| 통화    | 해당 조합 회계 통화를 확인할 수 있습니다.            |
| 계정명   | 각 보고서의 주요 계정명과 그 값을 확인할 수 있습니다.     |

#### 재무제표 상세 조회

- 재무제표 목록에서 특정 행을 클릭하면, 해당 조합 재무제표의 상세 정보를 확인할 수 있습니다.
- 혹은 `조합관리` > `조합 정보` 메뉴에서 `재무제표` 탭을 클릭하여 확인할 수도 있습니다.
![이미지](/assets/img/lfd105.jpg)

##### 기본정보 
- 상세 화면 상단의 **기본 정보 영역**에서는 다음 항목을 확인할 수 있습니다.

| 항목명    | 설명                                  |
| ------ | ----------------------------------- |
| 조합명    | 해당 재무제표의 대상 조합명을 확인할 수 있습니다.        |
| 운용사명   | 해당 조합의 운용사명을 확인할 수 있습니다.            |
| 회계기준   | 해당 조합 회계원장의 회계기준을 확인할 수 있습니다.       |
| 통화     | 해당 조합 회계 통화를 확인할 수 있습니다.            |
| 결산월    | 해당 조합 회계연도의 결산월을 확인할 수 있습니다.        |
| 결산여부   | 해당 기준일 조합 회계원장의 결산 여부를 확인할 수 있습니다.  |
| 회계감사기관 | 해당 조합 재무제표에 대한 회계감사기관명을 확인할 수 있습니다. |

##### 요약 재무제표 정보
- 주요 계정으로만 구성된 요약 재무상태표와 요약 손익계산서를 확인할 수 있습니다.
- 각 보고서의 기준일 및 기준기간도 확인할 수 있습니다.

##### 첨부파일
- 전체 계정으로 구성된 재무상태표와 손익계산서 파일을 클릭해 내려받을 수 있습니다.


## 자주 묻는 질문

> 조회하고자 하는 조합 재무제표의 기준일 정보가 없어요.
> {: .prompt-tip }

- 현재 LPworks에서는 **GP가 VCworks를 통해 전송한 조합 재무제표만 조회**할 수 있습니다.
- 기준일 선택란에는 수신된 조합 재무제표의 기준일만 등장합니다.
    

> 요약 재무제표 항목과 첨부파일에 첨부된 재무제표 보고서의 항목 값이 달라요.
> {: .prompt-tip }

- 요약 재무제표는 실제 조합 재무제표를 요약하여 보여주고 있습니다.
- 조회의 편의를 위하여 **특정 계정과목들은 요약 재무제표 항목에 포함되도록 분류**했습니다.
- 상세한 계정과목과 해당 값은 첨부파일로 확인하실 수 있습니다.
