투자 의무 상세 화면은 선택한 투자 의무의 기본 정보와 달성 현황을 확인하는 화면입니다.

투자 의무의 조건, 의무/한도 금액, 달성 금액, 달성률, 적용 기간, 규약 원문을 확인할 수 있으며, 해당 의무를 충족하는 투자 건 목록을 함께 조회할 수 있습니다.

## 설명

### 화면 경로

`LPworks` → `조합 관리` → `투자 의무 달성 현황` → `투자 의무 상세`

### 화면 구성

![](/assets/img/lfd311img.png)

### 기본정보 확인

기본정보 영역에서는 선택한 투자 의무의 주요 내용을 확인할 수 있습니다.

| 항목 | 설명 |
| --- | --- |
| 조합명 | 투자 의무가 설정된 조합명입니다. |
| 운용사명 | 해당 조합을 운용하는 운용사명입니다. |
| 투자 의무 | 투자 의무의 이름입니다. |
| 투자 의무 유형 | 투자 의무의 유형입니다. |
| 투자 조건 요약 | 투자 의무에 적용되는 조건을 요약하여 표시합니다. |
| 의무/한도 금액 | 투자 의무에 설정된 기준 금액입니다. |
| 달성 금액 | 해당 의무 기준으로 실제 투자 집행된 금액의 합계입니다. |
| 달성률 | 의무/한도 금액 대비 달성 금액의 비율입니다. |
| 의무 시작일 | 투자 의무가 적용되기 시작하는 날짜입니다. |
| 의무 종료일 | 투자 의무가 종료되는 날짜입니다. |
| 규약 원문 | 해당 투자 의무의 근거가 되는 규약 내용을 표시합니다. |

### 달성률 확인

달성률은 선택한 투자 의무가 현재 어느 정도 달성되었는지 보여주는 값입니다.

`달성 금액 ÷ 의무/한도 금액 × 100`으로 계산되며, 기본정보 영역에서 막대 그래프와 비율로 표시됩니다.

### 투자 건 목록 확인

투자 건 목록에서는 해당 투자 의무를 충족하는 투자 건을 확인할 수 있습니다.

| 컬럼 | 설명 |
| --- | --- |
| 포트폴리오 | 투자 건의 포트폴리오 이름입니다. |
| 투자일 | 투자 건의 투자일입니다. |
| 투자유형 | 투자 건의 투자 유형입니다. |
| 신주/구주 | 투자 건이 신주 투자 또는 구주 투자 중 어디에 해당하는지 표시합니다. |
| 투자 원금(조합 통화) | 투자 건의 투자 원금을 조합 통화 기준으로 표시합니다. |
| 비중 | 해당 투자 건의 투자 원금이 전체 의무/한도 금액에서 차지하는 비율입니다. |
| 비고 | 투자 건과 관련된 참고 사항이 표시됩니다. |

### 합계 확인

투자 건 목록 하단에서는 투자 원금과 비중의 합계를 확인할 수 있습니다.

비중은 각 투자 건의 투자 원금이 의무/한도 금액에서 차지하는 비율을 합산한 값입니다. 투자 건 구성에 따라 100% 이하일 수도 있고, 100%를 초과할 수도 있습니다.

### 검색 및 필터

각 컬럼 상단의 입력창 또는 필터 아이콘을 사용하여 투자 건 목록을 검색하거나 조건별로 필터링할 수 있습니다.

| 필터 유형 | 적용 컬럼 |
| --- | --- |
| 텍스트 검색 | 포트폴리오, 비고 |
| 날짜 검색 | 투자일 |
| 선택 필터 | 투자유형, 신주/구주 |
| 숫자 검색 | 투자 원금(조합 통화) |

### 포트폴리오 상세 이동

투자 건 목록에서 포트폴리오명을 클릭하면 해당 포트폴리오의 상세 정보 화면으로 이동할 수 있습니다.

## 자주 묻는 질문

> 이 화면에서는 무엇을 확인할 수 있나요?
{: .prompt-tip }

- 선택한 투자 의무의 조건, 달성 금액, 달성률, 적용 기간, 규약 원문과 해당 의무를 충족하는 투자 건을 확인할 수 있습니다.

> 달성률은 어떻게 계산되나요?
{: .prompt-tip }

- 달성률은 `달성 금액 ÷ 의무/한도 금액 × 100`으로 계산됩니다.

> 투자 건 목록에는 어떤 투자 건이 표시되나요?
{: .prompt-tip }

- 선택한 투자 의무의 조건을 충족하는 투자 건이 표시됩니다.

> 비중은 무엇을 의미하나요?
{: .prompt-tip }

- 비중은 해당 투자 건의 투자 원금이 전체 의무/한도 금액에서 차지하는 비율입니다.

> 비중 합계가 100%를 넘을 수도 있나요?
{: .prompt-tip }

- 네. 투자 원금이 의무/한도 금액을 초과하는 경우 비중 합계가 100%를 넘을 수 있습니다.

> 규약 원문은 무엇인가요?
{: .prompt-tip }

- 규약 원문은 해당 투자 의무의 근거가 되는 조합 규약 내용을 의미합니다.
