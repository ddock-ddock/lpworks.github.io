LPworks에서 **사용자(계정)별로 메뉴 권한을 설정**하는 방법을 설명합니다.

---
## 설명
- 법인 조합원은 각기 다른 **메뉴 권한**을 지니는 **역할**을 등록할 수 있습니다.
- **사용자(계정)별**로 다른 **역할을 부여**해 메뉴 권한을 설정할 수 있습니다.
- 한 명의 사용자에게 복수의 역할을 부여할 수 있습니다.

### 역할 등록

역할을 **등록**하는 방법은 다음과 같습니다.
![이미지](/assets/img/역할등록드로어.jpg)
1. LPworks에 로그인하여 `설정-역할/권한 관리` 메뉴로 이동해 주세요.
2. '역할 설정' 섹션에 있는 [+ 등록] 버튼을 클릭해주세요.
	- 최초 접속 시, '관리자' 역할만 존재합니다.
3. 역할 등록 드로어에서 `역할명`을 입력해주세요.
	- `설명`은 역할에 대한 설명입니다. 필요 시 입력해주세요.
	- `사용여부`를 끄게 되면, 다시 키기 전까지 해당 역할은 **없는 것으로 취급**합니다.
4. 역할 등록 드로어의 [등록] 버튼을 클릭해주세요.
	- 입력하신 내용으로 역할이 등록됩니다.

### 역할 수정

등록된 역할에 대해 다음 2가지를 **수정**할 수 있습니다.
- **메뉴 권한** 수정
- **기본 정보** 수정

#### 메뉴 권한 수정
특정 역할의 **메뉴 권한**을 수정하는 방법은 다음과 같습니다.
![이미지](/assets/img/메뉴권한수정.jpg)
1. '역할 설정' 섹션에서 수정하고자 하는 `역할명`을 클릭해주세요.
2. '권한 설정' 섹션에서 [수정] 버튼을 클릭해주세요.
3. '권한 설정' 섹션에서 [메뉴 권한]을 클릭해주세요.
4. 해당 역할에게 접근 권한을 부여할 메뉴명에 대해서만 **체크박스**를 선택해주세요.
5. '권한 설정' 섹션에서 [저장] 버튼을 클릭해주세요.
	- 해당 역할에 대한 설정한 메뉴 접근 권한이 저장됩니다.

#### 기본 정보 수정
특정 역할에 대한 '**역할 정보**'와 해당 역할을 부여할 '**사용자 목록**'을 수정할 수 있습니다.
![이미지](/assets/img/기본정보수정.jpg)
1. '역할 설정' 섹션에서 수정하고자 하는 `역할명`을 클릭해주세요.
2. '권한 설정' 섹션에서 [수정] 버튼을 클릭해주세요.
3. '권한 설정' 섹션에서 [기본 정보]를 클릭해주세요.
4. 해당 역할의 '**역할 정보**'를 수정해주세요.
	- 등록 시와 마찬가지로, `역할명`은 반드시 입력해야 합니다.
	- `사용여부`를 끄게 되면, 다시 키기 전까지 해당 역할은 없는 것으로 취급합니다.
5. 해당 역할을 부여할 '**사용자 목록**'을 수정해주세요.
	- `+` 버튼 클릭을 통해 사용자를 **추가**할 수 있습니다.
	- `휴지통` 버튼 클릭을 통해 사용자를 **제거**할 수 있습니다.
6. '권한 설정' 섹션에서 [저장] 버튼을 클릭해주세요.
	- 해당 역할에 대한 수정한 역할 정보 및 사용자 목록이 저장됩니다.


## 자주 묻는 질문

> `설정`에서 `역할/권한 관리` 메뉴가 보이지 않아요.
> {: .prompt-tip }

- 현재 **법인** 조합원인 경우에만 역할/권한 관리 기능을 **제공**하고 있습니다.
    

> 특정 메뉴의 메뉴 권한 편집이 불가해요.
> {: .prompt-tip }

- '**관리자**' 역할의 경우에는 `사용자 관리`와 `역할/권한 관리` 2개 메뉴에 대한 **권한 편집이 불가**합니다.
- 이는 사용자 정보와 역할/권한을 관리할 수 있는 최소한의 사용자를 확보하기 위한 조치입니다.
     

> 역할 사용 여부를 끌 수 없어요.
> {: .prompt-tip }

- 기본적으로 '**관리자**' 역할은 **사용 여부를 끌 수 없습니다**.
  (사유는 위 질문에 대한 사유와 동일합니다)
- 이외의 역할의 경우, **해당 역할만 부여 받은 사용자**가 존재하는 경우에 끌 수 없습니다.
	- 해당 사용자에 다른 역할도 부여하시면, 해당 역할의 사용 여부를 끌 수 있습니다.
    

> 사용자 목록에서 사용자 제거가 안돼요.
> {: .prompt-tip }

- **해당 역할만 부여 받은 사용자**일 경우에, 해당 역할의 사용자 목록에서 제거할 수 없습니다.
	- 해당 사용자에 다른 역할도 부여하시면, 해당 역할에서 사용자 제거가 가능합니다.
    

> 여러 개의 역할을 부여 받은 사용자는 어떤 권한을 지니나요?
> {: .prompt-tip }

- 한 명의 사용자가 복수의 역할을 부여 받았을 경우에는, 복수의 역할에 **부여된 모든 메뉴 권한**을 지닙니다.
    

> 신규 메뉴가 등록되면 모든 역할에 권한이 자동으로 부여되나요?
> {: .prompt-tip }

- 신규 메뉴가 등록되면, '**관리자**' 역할만 해당 메뉴에 대한 권한을 **자동으로 부여** 받습니다.
- 이외의 역할은 신규 메뉴에 대한 **권한 수정이 필요**합니다.
