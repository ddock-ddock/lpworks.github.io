업무집행조합원이 전송한 조합의 출자/배분 계획을 보고 기준일별로 조회하고, 해당 계획의 상세 정보를 조회할 수 있는 기능에 대한 설명입니다.

## 설명
- 출자/배분 계획에서는 조합원이 GP로부터 본인에게 전달된 **출자/배분 계획을 조회**하고, **해당 계획의 출자 요청일/배분 예정일별 상세 금액**을 확인할 수 있습니다.

#### 출자/배분 계획 목록 조회

조합원에게 전송된 출자/배분 계획 목록을 조회하는 방법은 다음과 같습니다.
![이미지](/assets/img/lp출배계메뉴얼1.jpg)

1. LPworks에 로그인해 주세요.
2. `조합-출자/배분-출자/배분 계획` 메뉴로 이동해 주세요.
3. 목록에서 본인에게 전달된 출자/배분 계획 내역을 조합-보고 기준일별로 확인할 수 있습니다.
  
목록에서는 다음 정보를 확인할 수 있습니다.

| 항목명      | 설명                                 |
| -------- | ---------------------------------- |
| 보고 기준일      | 출자/배분 계획의 보고 기준일을 확인할 수 있습니다.           |
| 구분       | 계획의 구분을 확인할 수 있습니다(출자 혹은 배분).             |
| 조합명   | 출자/배분 계획의 대상이 되는 조합명을 확인할 수 있습니다.          |
| 운용사명       | 해당 계획의 대상 조합의 운용사명을 확인할 수 있습니다. |
| 통화    | 출자/배분 계획 금액에 대한 통화를 확인할 수 있습니다.             |
| 총 계획금액     | 해당 출자/배분 계획에 할당된 총 계획 금액을 확인할 수 있습니다.           |


#### 출자/배분 계획 상세 조회

출자/배분 계획 목록에서 특정 행을 클릭하면, 해당 출자/배분 계획의 일자별 상세 정보를 확인할 수 있습니다.
![이미지](/assets/img/lp출배계메뉴얼2.jpg)

###### 기본정보 확인 
상세 화면 상단의 **기본 정보 영역**에서는 다음 항목을 확인할 수 있습니다.

| 항목명   | 설명                                  |
| ----- | ----------------------------------- |
| 구분   | 계획의 구분입니다(출자 혹은 배분).            |
| 보고 기준일    | 출자/배분 계획의 보고 기준일입니다.              |
| 운용사명    | 해당 계획의 대상 조합의 운용사명입니다.           |
| 통화 | 출자/배분 계획 금액에 대한 통화입니다.            |
| 조합명   | 출자/배분 계획의 대상이 되는 조합명입니다.                  |
| 조합원명 | 조회하시는 조합원명입니다. |
| 조합 전체 약정 금액 | 조합의 전체 약정 금액입니다. |
| 조합 전체 기 납입액 | 현재까지 조합에 누적된 출자 금액입니다. |
| 조합 전체 약정 잔액 | 조합 전체 약정 금액에서 기 납입액을 차감한 잔여 출자 금액입니다. |
| 조합 전체 기 배분액 | 현재까지 조합에 누적된 배분 총액입니다. |
| 약정 금액 | 해당 조합에 대한 조합원의 약정 금액입니다. |
| 기 납입액 | 현재까지 조합에 누적된 조합원의 출자 금액입니다. |
| 약정 잔액 | 조합원의 약정 금액에서 조합원의 기 납입액을 차감한 잔여 출자 금액입니다. |
| 기 배분액 | 현재까지 조합원에게 누적된 배분 총액입니다. |

###### 출자 요청/배분 계획 확인
기본 정보 하단의 **출자 요청/배분 계획 영역**에서는 해당 계획의 출자 요청일/배분 예정일별 예정 금액 상세를 확인할 수 있습니다.

| 항목명      | 설명                                |
| -------- | --------------------------------- |
| 출자 요청일 | 출자 요청이 계획된 일자입니다.          |
| 배분 예정일 | 배분이 예정된 일자입니다.          |
| 조합 전체 요청 금액  | 해당 계획 일자에 조합에 요청되는 출자 금액입니다.     |
| 조합 전체 배분 총액  | 해당 계획 일자에 조합에 배분되는 배분 총액입니다.     |
| 요청 금액     | 해당 계획 일자에 조합원에 요청되는 출자 금액입니다.            |
| 배분 원금     | 해당 계획 일자에 조합원에 요청되는 배분 원금입니다.            |
| 배분 수익     | 해당 계획 일자에 조합원에 요청되는 배분 수익입니다.            |
| 배분 총액     | 해당 계획 일자에 조합원에 요청되는 배분 총액(배분 원금 + 배분 수익)입니다.            |
| 출자 요청 사유     | 해당 계획 일자에 출자가 요청되는 사유입니다.                   |
| 배분 사유     | 해당 계획 일자에 배분이 예정되는 사유입니다.                   |


###### 첨부파일
- 운용사가 출자/배분 계획과 함께 등록한 첨부파일이 있는 경우, 상세 화면 하단에서 확인할 수 있습니다.
- 첨부파일은 다운로드하여 열람할 수 있습니다.
- 첨부파일이 없는 경우, **“첨부파일이 없습니다.”**로 표시됩니다.


## 자주 묻는 질문

> 출자/배분 계획과 출자/배분은 무슨 차이가 있나요?
> {: .prompt-tip }

- 출자/배분 계획은 말 그대로 운용사가 입력한 미래의 출자/배분 계획 정보입니다.
- 보고 기준일보다 미래의 출자/배분 계획만 포함하고 있습니다.
- 실제로 발생하는 출자/배분 내역과는 차이가 있을 수 있습니다.
    

> 실제 납입했던 금액과 특정 계획의 '기본정보' 영역에서 확인되는 `기 납입액`/`기 배분액`이 달라요. 
> {: .prompt-tip }

- 출자/배분 계획 상세의 '기본정보' 영역에서 확인되는 `기 납입액`과 `기 배분액`은 운용사가 해당 계획을 작성하는 시점에 조회하여 등록된 값입니다.
- 계획을 등록한 후에 실제 출자/배분이 발생한 경우, **실제 기 납입액/기 배분액과** 위에서 조회하신 값은 **다를 수 있습니다.**
    

> 첨부 파일이 보이지 않아요.  
> {: .prompt-tip }

- 운용사가 첨부파일을 등록하지 않은 경우, 파일이 표시되지 않을 수 있습니다.
- 필요한 자료가 없는 경우 운용사에 문의해 주세요.
