LPworks 알림 화면에서는 조합원총회, 출자금 납입 요청, 문서 발급, 파일 다운로드 등 사용자가 확인해야 할 중요한 정보를 한 곳에서 확인할 수 있습니다.  
알림을 통해 관련 화면으로 이동하거나 파일을 다운로드할 수 있으며, 알림을 읽음 처리, 보관, 삭제하여 관리할 수 있습니다.

---
## 설명

![이미지](/assets/img/alarm.png)
### 1. 알림 화면 구성
알림 화면은 사용자가 수신한 알림을 조회하고 관리할 수 있는 화면입니다.

#### (1) 알림 탭
알림은 다음 3개의 탭으로 구분됩니다.

- **안읽음**: 아직 확인하지 않은 알림
- **전체**: 모든 알림
- **보관함**: 사용자가 보관한 알림

#### (2) 검색 및 필터
알림을 빠르게 찾기 위해 검색 및 필터 기능을 사용할 수 있습니다.

- 제목 / 내용 검색
- 알림 유형 필터
- 출처 메뉴 필터

---

### 2. 알림 목록
알림은 카드 형태로 표시되며, 수신일시 기준 최신순으로 정렬됩니다.

알림 카드에는 다음 정보가 표시됩니다.

- 알림 제목
- 알림 내용
- 수신일
- 읽음 / 보관 / 삭제 버튼

알림 카드에서 수행할 수 있는 작업은 다음과 같습니다.

| 작업 | 설명 |
|------|------|
| 알림 클릭 | 관련 화면으로 이동하거나 파일 다운로드 |
| 읽음 처리 | 알림을 읽음 상태로 변경 |
| 보관 | 알림을 보관함으로 이동 |
| 삭제 | 알림 삭제 |

여러 개의 알림을 선택하여 일괄로 읽음 처리, 보관, 삭제할 수도 있습니다.

---

### 3. 파일 다운로드
파일 다운로드가 필요한 알림의 경우, 알림 클릭 시 다운로드가 진행됩니다.

- 파일이 1개인 경우 → 즉시 다운로드
- 파일이 여러 개인 경우 → 다운로드 모달에서 개별 다운로드 또는 일괄 다운로드
- 다운로드는 파일 유효기간 내에서만 가능

※ 파일 다운로드 유효기간은 **알림 수신 후 30일**입니다.

---

### 4. 알림 상태 관리
사용자는 알림을 다음과 같이 관리할 수 있습니다.

#### (1) 읽음 / 미읽음 처리
- 알림을 클릭하면 읽음 처리됩니다.
- 선택 후 읽음/미읽음 버튼으로 상태 변경 가능

#### (2) 보관
- 중요한 알림은 보관함에 저장할 수 있습니다.
- 보관된 알림은 자동 삭제되지 않습니다.

#### (3) 삭제
- 알림 삭제 시 복구할 수 없습니다.

---

### 5. 알림 유형
LPworks에서는 다음과 같은 유형의 알림이 발송됩니다.

| 알림 유형 | 설명 |
|-----------|------|
| 정보 알림 | 일반 정보 안내 |
| 요청 | 사용자의 액션이 필요한 알림 |
| 파일 다운로드 | 파일 다운로드 안내 |
| 문서 발급 | 증명서 등 문서 발급 안내 |
| 서비스 안내 | 신규 기능 안내 |

대표적인 알림 예시는 다음과 같습니다.

- 조합원총회 개최 안내
- 조합원총회 결과 안내
- 출자금 납입 요청
- 출자/배분 계획 안내
- 소득공제 문서 안내
- 투자확인서 다운로드
- 신규 기능 안내
- 증명서 발급 완료 안내

---

### 6. 알림 유효기간
알림은 수신일 기준으로 자동 관리됩니다.

| 항목 | 정책 |
|------|------|
| 알림 보관 안함 | 30일 후 자동 삭제 |
| 알림 보관함 | 삭제되지 않음 |
| 파일 다운로드 | 30일 이후 다운로드 불가 |

---

### 7. 모바일 알림
모바일에서는 일부 알림만 제공됩니다.

- 읽지 않은 알림 / 지난 알림으로 구분
- 30일 이상 지난 알림은 자동 삭제
- 모바일에서는 알림 보관 불가
- 모바일에서는 현재 조합원총회 알림만 제공됩니다.

## 자주 묻는 질문

> 알림을 삭제하면 다시 확인할 수 있나요?
{: .prompt-tip }

- 알림을 삭제하면 복구할 수 없습니다. 중요한 알림은 보관 기능을 이용하시기 바랍니다.

> 파일 다운로드가 되지 않습니다.
{: .prompt-tip }

- 파일 다운로드는 알림 수신 후 30일 이내에만 가능합니다. 다운로드 기간이 만료된 경우 파일을 다시 요청해야 합니다.

> 읽지 않은 알림은 어디에서 확인할 수 있나요?
{: .prompt-tip }

- 알림 화면 상단의 **안읽음 탭**에서 확인할 수 있습니다.
