조합정보의 투자 의무 탭은 선택한 조합의 투자 의무와 달성 현황을 확인하는 화면입니다.

투자 건별로 각 투자 의무에 해당하는지 확인할 수 있으며, 조합에 설정된 투자 의무별 달성률, 의무/한도 금액, 달성 금액, 잔여 금액을 함께 조회할 수 있습니다.

## 설명

### 화면 경로

`LPworks` → `조합 관리` → `조합 정보` → `투자 의무`

### 화면 구성

![[Pasted image 20260604145712.png]]

#### 투자 의무 화면 전환

투자 의무 탭은 2개의 보기 방식으로 구성되어 있습니다.

| 구분        | 설명                                   |
| --------- | ------------------------------------ |
| 투자건별 의무현황 | 조합이 투자한 각 투자 건이 투자 의무별로 해당되는지 확인합니다. |
| 의무 현황     | 조합에 설정된 투자 의무별 달성 현황을 확인합니다.         |

## 투자건별 의무현황 확인

`투자건별 의무현황`에서는 투자 건 단위로 투자 의무 해당 여부를 확인할 수 있습니다.

각 행은 하나의 투자 건을 의미하며, 투자 의무 수만큼 의무 컬럼이 생성됩니다.

| 컬럼 | 설명 |
| --- | --- |
| 포트폴리오 | 투자 건의 포트폴리오 이름입니다. |
| 투자일 | 투자 건의 투자일입니다. |
| 투자유형 | 투자 건의 투자 유형입니다. |
| 신주/구주 | 투자 건이 신주 투자 또는 구주 투자 중 어디에 해당하는지 표시합니다. |
| 투자원금(조합통화) | 투자 건의 투자 원금을 조합 통화 기준으로 표시합니다. |
| 투자 의무 유형 | 해당 투자 건이 투자 의무에 해당하면 `여`, 해당하지 않으면 `부`로 표시됩니다. |

투자 의무 유형 컬럼의 값을 클릭하면 해당 투자 의무 상세 화면으로 이동할 수 있습니다.

### 의무 현황 확인

![[Pasted image 20260604162049.png]]

`의무 현황`에서는 조합에 설정된 투자 의무별 달성 현황을 확인할 수 있습니다.

이 목록은 `투자 의무 달성 현황` 화면에서 해당 조합으로 필터링한 결과와 동일한 기준으로 표시됩니다.

| 컬럼 | 설명 |
| --- | --- |
| 투자의무 유형 | 투자 의무의 유형입니다. |
| 의무명 | 투자 의무의 이름입니다. 의무명을 클릭하면 투자 의무 상세 화면으로 이동합니다. |
| 조건 | 하나의 투자 의무에 여러 조건이 있는 경우 조건 정보가 표시됩니다. |
| 달성률 | 투자 의무의 달성 비율입니다. VCworks에서 전달받은 값을 표시하거나, 달성 금액과 의무/한도 금액을 기준으로 계산합니다. |
| 의무/한도 금액 | 투자 의무에 설정된 기준 금액입니다. |
| 달성 금액 | 해당 투자 의무에 해당하는 투자 건들의 투자금 합계입니다. |
| 잔여 금액 | 추가로 달성하거나 관리해야 하는 금액입니다. |
| 시작일 | 투자 의무 적용 시작일입니다. |
| 종료일 | 투자 의무 적용 종료일입니다. |

### 달성률 확인

달성률은 투자 의무가 현재 어느 정도 충족되었는지 보여주는 값입니다.

일반적으로 `달성 금액 ÷ 의무/한도 금액 × 100`으로 계산됩니다.

### 투자 의무 상세 이동

투자 의무 상세 화면은 다음 항목에서 이동할 수 있습니다.

| 위치 | 동작 |
| --- | --- |
| 투자건별 의무현황 | 투자 의무 유형 컬럼의 `여` 또는 `부` 값을 클릭합니다. |
| 의무 현황 | 의무명을 클릭합니다. |

투자 의무 상세 화면에서는 해당 의무의 기본정보, 규약 원문, 의무를 충족하는 투자 건 목록을 확인할 수 있습니다.

### 검색 및 필터

각 목록의 컬럼 상단 입력창 또는 필터 아이콘을 사용하여 원하는 투자 건이나 투자 의무를 빠르게 찾을 수 있습니다.

| 보기 방식 | 검색 및 필터 대상 |
| --- | --- |
| 투자건별 의무현황 | 포트폴리오, 투자일, 투자유형, 신주/구주, 투자원금, 투자 의무 해당 여부 |
| 의무 현황 | 투자의무 유형, 의무명, 조건, 달성률, 의무/한도 금액, 달성 금액, 잔여 금액, 시작일, 종료일 |

### 표시 기준

- `투자건별 의무현황`은 조합의 투자 건을 기준으로 표시됩니다.
- `의무 현황`은 조합에 설정된 투자 의무를 기준으로 표시됩니다.
- 하나의 투자 의무에 조건이 여러 개인 경우, 조건별로 구분하여 표시될 수 있습니다.
- 투자 의무 노출 범위는 운용사가 설정한 공개 기준에 따라 달라질 수 있습니다.

## 자주 묻는 질문

> 이 화면에서는 무엇을 확인할 수 있나요?
{: .prompt-tip }

- 선택한 조합의 투자 건별 투자 의무 해당 여부와 투자 의무별 달성 현황을 확인할 수 있습니다.

> 투자건별 의무현황과 의무 현황은 어떤 차이가 있나요?
{: .prompt-tip }

- 투자건별 의무현황은 투자 건을 기준으로 각 의무에 해당하는지 보여주고, 의무 현황은 투자 의무를 기준으로 달성률과 금액을 보여줍니다.

> 투자 의무 유형 컬럼의 `여`와 `부`는 무엇인가요?
{: .prompt-tip }

- `여`는 해당 투자 건이 투자 의무 조건에 해당한다는 의미이고, `부`는 해당하지 않는다는 의미입니다.

> 의무 현황은 투자 의무 달성 현황 목록과 같은 정보인가요?
{: .prompt-tip }

- 네. 의무 현황은 `투자 의무 달성 현황` 화면에서 현재 조합으로 필터링한 결과와 같은 기준으로 표시됩니다.

> 투자 의무 상세 화면으로 이동하려면 어떻게 하나요?
{: .prompt-tip }

- 투자건별 의무현황에서는 투자 의무 해당 여부 값을 클릭하고, 의무 현황에서는 의무명을 클릭하면 투자 의무 상세 화면으로 이동합니다.

> 하나의 투자 의무에 조건이 여러 개 있으면 어떻게 표시되나요?
{: .prompt-tip }

- 조건별로 구분하여 표시될 수 있습니다. 이 경우 조건 컬럼에서 각 조건 정보를 확인할 수 있습니다.

> 모든 조합원이 투자 의무 정보를 볼 수 있나요?
{: .prompt-tip }

- 투자 의무 정보의 노출 범위는 운용사가 설정한 공개 기준에 따라 달라질 수 있습니다.
